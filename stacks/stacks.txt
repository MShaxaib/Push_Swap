



/* How to make stacks */


Step 1: define a structure with data and next
Step 2: Malloc for the structure
Step 3: define a node pointer HEAD and set it to NULL


How to PUSH content into the stack

Step 1: create a node with the value
Step 2: Check if the head of the stack is empty
Step 3: if EMPTY :  set newnode->next = NULL
        if !EMPTY : set newnode->next = HEAD
Step 4: set Head = newnode


How to POP

Step 1: Check if the stack is empty
Step 2: Define a node pointer 'temp' and set it to HEAD
Step 3: Set HEAD = HEAD -> next
Step 4: Delete and free temp
